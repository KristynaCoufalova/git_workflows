# General Git Flow Structure Diagram
+----------------------------+
|      Deployable master     |
|          Branch            |
+----------------------------+
              |
              v
+----------------------------+
|     Create Develop Branch  |
+----------------------------+
              |
              v
+----------------------------+
|   Create Feature Branch    |
+----------------------------+
              |
              v
+----------------------------+
|  Commit & Push Changes     |
+----------------------------+
              |
              v
+----------------------------+
|   Open Pull Request (PR)   |
+----------------------------+
              |
              v
+----------------------------+
|   Merge Feature into       |
|        Develop Branch      |
+----------------------------+
              |
              v
+----------------------------+
|   Create Release Branch    |
+----------------------------+
              |
              v
+----------------------------+
|   Test & Prepare Release   |
+----------------------------+
              |
              v
+----------------------------+
|    Merge Release into      |
|          Master Branch     |
+----------------------------+
              |
              v
+----------------------------+
|  Deploy to Production      |
+----------------------------+
